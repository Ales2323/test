## 🧪 Complete Testing Workflow:

### 1. QR Code Generation:
1. Go to: https://ales2323.github.io/test/qr-generator.html
2. **Custom QR**: Select service type and enter location
3. **Quick Generate**: Select from preset options
4. Download or print QR codes for placement

### 2. Customer Ordering Experience:
1. **Scan QR Code** or visit direct URL
2. **Enter Guest Information**: Room number, name, phone
3. **Browse Menu**: Navigate categories and subcategories
4. **Add Items**: Use quantity controls and add to cart
5. **Checkout**: Review order and submit payment details
6. **Confirmation**: Receive order number and delivery time

### 3. Admin Order Management:
1. Go to: https://ales2323.github.io/test/admin.html
2. **View Orders**: Real-time order display with filtering
3. **Update Status**: Pending → Preparing → Ready → Delivered
4. **Menu Management**: Add/edit menu items and prices
5. **Generate Reports**: View sales analytics and metrics
